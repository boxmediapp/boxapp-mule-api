After confirming the identity of the user, you can approve the user.

You can do so by assignning a role to the new user in the user management console, which is in the Admin section.

The new user is given the 'user' role when he/she has subsribed to the system. The 'user' role does not have permission to use the system. 

If the new user is an image client, you should assign the 'image client' role to the new user.

If the new user is an Box Plus operator, you should assign the "operator" role to the user.

If the new user is an admin, you should assing the 'admin' role to the user.
